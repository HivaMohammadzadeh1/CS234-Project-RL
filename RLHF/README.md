# RLHF Implementation

This folder contains the complete implementation of Reinforcement Learning from Human Feedback (RLHF) for video generation models.

## Notebooks

- **[Final]\_RLHF_Reward_Model_Training.ipynb**: Notebook for training the reward model using our human annotation data.
- **[Final]\_RLHF_PPO_Training.ipynb**: Notebook for PPO (Proximal Policy Optimization) training, adapted to work with diffusion models.
- **Baseline_Videos.ipynb**: Notebook for generating baseline videos using the original model.
- **Generate_RLHF_Evaluation_Videos.ipynb**: Notebook for generating evaluation videos using the fine-tuned RLHF model.

## Directories

- **eval_vids/**: Contains all videos generated by the new RLHF-trained model for evaluation purposes.
- **final_checkpoints/**: Stores the final trained model checkpoints.
  - **reward-model-final/**: Checkpoints for the trained reward model.
  - **wan-ppo-final/**: Checkpoints for the PPO-trained model with specific adaptations for diffusion models.
