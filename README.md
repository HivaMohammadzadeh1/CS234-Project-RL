# CS234 Project: Physics Video Generation and Ranking

This repository contains a video ranking system for evaluating AI-generated physics simulations, for tuning video models with reinforcement learning.

## Quick Start

### Video Generation

To generate new physics simulation videos:

1. Navigate to the generation folder:
   ```bash
   cd video_generation
   ```

2. See [video_generation/README.md](video_generation/README.md) for detailed instructions on:
   - Setting up the Wan2.2 diffusion model
   - Creating prompts for physics scenarios
   - Batch generating videos via fal.ai API

### Video Ranking

To rank and evaluate generated videos:

1. Install dependencies:
   ```bash
   pip install -r requirements_rater.txt
   ```

2. Run the video ranking server:
   ```bash
   python video_rater_server.py
   ```

3. Open your browser to: http://localhost:5001
